# SeniorCare Connect

A community-focused healthcare application designed to help senior citizens manage medicines, emergency contacts, health profiles, and AI-powered document reading.

## Features
- Medicine Reminders
- Emergency SOS
- QR Health Profile
- AI Medicine Reader
- Voice Assistance
- Accessibility Features

## Tech Stack
- HTML
- CSS
- JavaScript
- Vite
- Supabase
- Gemini API
